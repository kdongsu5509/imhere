## 주요 기능

*`발생 가능한 예외`도 함께 관리하고 있습니다.*

### Kotlin/Spring

- 회원가입/로그인
  - `Spring Security` 를 이용하여 `Oauth2` 를 사용한다.
  - `oidc` 를 이용한다.
  - `카카오` 를 제공자로 사용한다.

### Dart/Flutter

*Figma 화면 구성에 따른 안내*

- Figma에서의 화면 구성에서는 구성 요소들의 전반적인 배치만 참고.
- 디테일에는 우선 집착하지 않음.

- 회원가입/로그인
  - 화면을 기획에 맞게 구성한다
    - 변동 -> 구글 로그인을 MVP에서 사용하지 않기로 결정하면서 일부 변동.
  - 카카오 SDK를 이용하여 서버와 함께 정상작동하도록 구현한다
- 지오펜스
  - 화면
    - 화면1. 등록되어 있는 지오펜스를 볼 수 있는 View를 만든다.
    - 화면2. 지오펜스를 등록할 수 있는 View를 만든다.
- 연락처
  - [CONTACT_001] : 등록되어 있는 연락처를 볼 수 있는 화면
    - 나의 휴대전화에 있는 연락처에서 하나의 연락처를 불러온다.
      - Android[ ] , IOS[ ]
- 기록
  - 기록을 볼 수 있는 View를 만든다.
